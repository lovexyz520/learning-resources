# npm 教學資源

## 主要文件

- [`complete-guide.md`](complete-guide.md)：npm 從入門到實務的完整教學。
- [`package-json-reference.md`](package-json-reference.md)：`package.json` 欄位速查與常見模板。
- `troubleshooting.md`：常見安裝與版本衝突排查（規劃中）。

## 文件重點

- 了解 npm 與 Node.js 的關係與角色分工。
- 學會 `package.json`、`package-lock.json`、`node_modules` 的實際用途。
- 掌握依賴安裝策略：`dependencies`、`devDependencies`、`peerDependencies`。
- 建立可協作的版本管理流程：`engines`、`.nvmrc`、`npm ci`。
- 具備日常維運能力：升級、稽核、發佈、排錯。

## 建議閱讀順序

1. 先讀 `complete-guide.md` 的第 1 到 6 章，建立核心觀念。
2. 讀 `package-json-reference.md`，建立欄位與模板對照表。
3. 再讀 `complete-guide.md` 第 7 到 10 章，完成腳本與團隊協作流程。
4. 最後讀第 11 到 15 章，處理發佈、安全與常見問題。
